## 常見問題

### 問題 1：找不到 python 命令

**解決方案**：
- 確認已安裝 Python
- 嘗試使用 `python3` 代替 `python`
- 或者使用完整路徑，例如：`C:\Python39\python.exe`

### 問題 2：缺少模組（ModuleNotFoundError）

**解決方案**：

如果是 Flask 相關錯誤：
```bash
pip install Flask flask-cors
```

如果只是運行簡化版本，不需要安裝任何額外模組（只使用 Python 標準庫）。

### 問題 3：端口 5000 被占用

**解決方案**：
- 關閉其他使用 5000 端口的應用
- 或修改 `app.py` 中的端口號：
  ```python
  app.run(debug=True, host='0.0.0.0', port=5001)  # 改成 5001
  ```

### 問題 4：瀏覽器無法連接

**解決方案**：
1. 確認伺服器正在運行（終端應該顯示 "Running on http://..."）
2. 檢查防火牆設置
3. 嘗試訪問 `http://127.0.0.1:5000` 代替 `localhost:5000`
