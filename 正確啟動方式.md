# 正確啟動方式說明

## 問題診斷

您目前看到的命令提示字元是 **命令行互動版本**，不是 Web 伺服器。

## 正確的啟動步驟

### 步驟 1：關閉當前的命令提示字元

如果命令提示字元在等待輸入，請關閉它。

### 步驟 2：確認文件位置

確保您在「人類圖」資料夾中，應該有 `app.py` 文件。

### 步驟 3：啟動 Flask 伺服器（正確的方式）

**方法 A：使用批次檔**

雙擊 `啟動伺服器.bat`

**方法 B：使用命令提示字元**

1. 打開新的命令提示字元窗口
2. 切換到資料夾：
   ```
   cd "C:\Users\a0929\OneDrive\桌面\CURSOR\人類圖"
   ```
3. 運行：
   ```
   python app.py
   ```

### 步驟 4：確認伺服器啟動成功

**正確啟動時，您應該看到：**

```
============================================================
人類圖計算器 API 伺服器
============================================================

伺服器正在啟動...
訪問地址: http://localhost:5000

API 端點:
  GET  /          - 前端頁面
  POST /calculate_hd - 計算人類圖數據
  GET  /health    - 健康檢查

按 Ctrl+C 停止伺服器

============================================================
 * Running on http://0.0.0.0:5000
 * Debug mode: on
WARNING: This is a development server...
```

**關鍵識別點：**
- ✅ 看到 "Running on http://0.0.0.0:5000"
- ✅ 看到 "Debug mode: on"
- ❌ 不會詢問您輸入年份、月份等

### 步驟 5：在瀏覽器中測試

1. 保持伺服器運行（不要關閉命令提示字元）
2. 在瀏覽器中：
   - 如果已打開 `index.html`，直接點擊「計算人類圖」
   - 或訪問：`http://localhost:5000`

## 常見錯誤

### ❌ 錯誤：運行的是 `human_design_simplified.py`

這個文件會顯示：
```
請輸入出生年份 (YYYY, 例如: 1990): _
```

這是命令行版本，不是 Web 伺服器！

### ✅ 正確：運行的是 `app.py`

這個文件會顯示：
```
Running on http://0.0.0.0:5000
```

這是 Web 伺服器！

## 快速檢查清單

- [ ] 在「人類圖」資料夾中
- [ ] 運行的是 `app.py`（不是 `human_design_simplified.py`）
- [ ] 已安裝 Flask（`pip install Flask flask-cors`）
- [ ] 看到 "Running on http://0.0.0.0:5000" 訊息
- [ ] 伺服器窗口保持打開
- [ ] 瀏覽器中點擊「計算人類圖」按鈕

## 如果還是無法連接

1. **檢查端口**：確認沒有其他程序使用 5000 端口
2. **檢查防火牆**：可能需要允許 Python 通過防火牆
3. **嘗試訪問**：直接在瀏覽器輸入 `http://localhost:5000` 看看能否載入頁面
