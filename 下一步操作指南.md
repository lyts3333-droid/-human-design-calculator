## 📋 接下來的步驟

### 步驟 1：在項目資料夾中初始化 Git（已準備好腳本）

**方法 B：手動執行命令**

1. 打開命令提示符
2. 進入項目資料夾：


3. 執行以下命令：
   ```bash
   git init
   git add .
   git commit -m "準備 Netlify 部署"
   ```

### 步驟 2：創建 GitHub 帳號和倉庫

1. **訪問 GitHub**：https://github.com
2. **如果還沒有帳號**：
   - 點擊「Sign up」註冊
   - 填寫用戶名、郵箱、密碼
   - 完成註冊

3. **如果已有帳號**：
   - 直接登入

4. **創建新倉庫**：
   - 登入後，點擊右上角的 **「+」** 圖標
   - 選擇 **「New repository」**
   - 填寫倉庫信息：
     - **Repository name**: `human-design-calculator`（或任何您喜歡的名稱）
     - **Description**: （可選）人類圖計算器
     - **Visibility**: 選擇 **Public**（公開，免費）或 **Private**（私有，需要付費）
     - **重要**：不要勾選「Initialize this repository with a README」
   - 點擊 **「Create repository」**

---

### 步驟 3：連接並推送代碼到 GitHub

GitHub 創建倉庫後會顯示設置說明，但基本步驟是：

1. **在命令提示符中執行**（將 `YOUR_USERNAME` 和 `YOUR_REPO_NAME` 替換為您的實際值）：

   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git branch -M main
   git push -u origin main
   ```

2. **如果要求輸入帳號密碼**：
   - 用戶名：您的 GitHub 用戶名
   - 密碼：使用 **Personal Access Token**（不是 GitHub 登入密碼）
   - **獲取 Token 的方法**：
     1. 登入 GitHub
     2. 點擊右上角頭像 → **Settings**
     3. 左側選單 → **Developer settings**
     4. 選擇 **Personal access tokens** → **Tokens (classic)**
     5. 點擊 **Generate new token** → **Generate new token (classic)**
     6. 填寫信息：
        - Note: `Netlify Deploy`
        - 選擇過期時間（建議選擇較長時間）
        - 勾選 **repo**（完整控制私有倉庫）
     7. 點擊 **Generate token**
     8. **複製 Token**（只顯示一次，請保存好）
     9. 在命令提示符中，密碼處貼上這個 Token

---

### 步驟 4：在 Netlify 連接 Git 倉庫

1. **登入 Netlify**：https://app.netlify.com

2. **添加新網站**：
   - 點擊 **「Add new site」**
   - 選擇 **「Import an existing project」**

3. **連接 Git 提供者**：
   - 選擇 **GitHub**（或您使用的 Git 提供者）
   - 如果是第一次，會要求授權，點擊 **「Authorize Netlify」**
   - 確認授權範圍，點擊 **「Authorize」**

4. **選擇倉庫**：
   - 在倉庫列表中，找到並選擇您剛創建的倉庫
   - 點擊它

5. **配置構建設置**：
   - Netlify 會自動檢測 `netlify.toml` 配置文件
   - 確認設置：
     - Build command: （自動從 netlify.toml 讀取）
     - Publish directory: `.`
     - Functions directory: `netlify/functions`
   - 如果都正確，點擊 **「Deploy site」**

6. **等待部署**：
   - 通常需要 1-2 分鐘
   - 等待看到 **「Deploy successful!」** 或 **「Published」**

---

### 步驟 5：訪問您的網站

部署完成後：

1. Netlify 會提供一個網址，例如：
   ```
   https://your-site-name-12345.netlify.app
   ```

2. 點擊這個網址訪問您的網站

3. 測試功能：
   - 輸入出生資料
   - 點擊「計算人類圖」
   - 應該可以正常運作

## 🆘 如果遇到問題

**問題 1：無法推送到 GitHub**
- 確認 GitHub 倉庫 URL 正確
- 確認使用 Personal Access Token 作為密碼

**問題 2：Netlify 找不到倉庫**
- 確認已授權 Netlify 訪問 GitHub
- 確認倉庫名稱正確

**問題 3：部署失敗**
- 查看 Netlify 構建日誌
- 確認 `netlify.toml` 配置正確
