# 安裝指南

## 📦 安裝步驟

### 方法 1：使用批處理文件（推薦，Windows）

#### 快速安裝（只安裝依賴）
1. 雙擊運行 `install.bat`
2. 等待安裝完成
3. 安裝完成後，運行 `start_server.bat` 或 `python app.py` 啟動服務器

#### 完整安裝（安裝依賴並啟動服務器）
1. 雙擊運行 `完整安裝.bat`
2. 等待安裝完成
3. 服務器會自動啟動

### 方法 2：手動安裝（所有系統）

#### 步驟 1：檢查 Python 版本
```bash
python --version
```
需要 Python 3.7 或更高版本

#### 步驟 2：安裝依賴
```bash
pip install -r requirements.txt
```

或者逐個安裝：
```bash
pip install Flask==3.0.0
pip install flask-cors==4.0.0
pip install flask-login==0.6.3
pip install flask-sqlalchemy==3.1.1
pip install werkzeug==3.0.1
pip install pyswisseph
pip install pytz==2024.1
pip install gunicorn
pip install pandas
```

#### 步驟 3：啟動服務器
```bash
python app.py
```

#### 步驟 4：訪問應用
打開瀏覽器，訪問：`http://localhost:5000`

## 🔧 新功能說明

### 新增的依賴包
- **flask-login** - 用戶登錄認證
- **flask-sqlalchemy** - 數據庫 ORM
- **werkzeug** - 密碼哈希和安全工具

### 數據庫
- 首次運行時會自動創建 `human_design.db` 數據庫文件
- 數據庫存儲用戶信息和歷史記錄
- 無需手動配置

## ⚠️ 常見問題

### 問題 1：pip 安裝失敗
**解決方法：**
```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### 問題 2：找不到 Python
**解決方法：**
- 確保已安裝 Python 3.7+
- 將 Python 添加到系統 PATH
- 或使用完整路徑：`C:\Python3x\python.exe app.py`

### 問題 3：模塊導入錯誤
**解決方法：**
- 確認所有依賴都已安裝：`pip list`
- 重新安裝：`pip install -r requirements.txt --force-reinstall`

### 問題 4：端口被占用
**解決方法：**
- 修改 `app.py` 最後一行的端口號
- 或關閉占用 5000 端口的程序

## 📝 驗證安裝

安裝完成後，運行以下命令驗證：
```bash
python -c "import flask, flask_login, flask_sqlalchemy, werkzeug; print('所有依賴已安裝！')"
```

如果沒有錯誤，表示安裝成功！

## 🚀 開始使用

1. 啟動服務器：`python app.py`
2. 打開瀏覽器訪問：`http://localhost:5000`
3. 首次使用需要註冊帳號
4. 登錄後即可使用所有功能
