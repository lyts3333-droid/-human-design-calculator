================================================================================
                    Netlify 快速部署指南（一步一步）
================================================================================

【步驟 1】準備文件（確認這些文件存在）
--------------------------------------------------------
✓ netlify.toml（在根目錄）
✓ index_netlify.html（在根目錄）
✓ netlify/functions/calculate_hd/__init__.py
✓ netlify/functions/requirements.txt
✓ .gitignore（已配置好，會自動排除不需要的文件）


【步驟 2】登入 Netlify
--------------------------------------------------------
1. 打開瀏覽器，訪問：https://app.netlify.com
2. 點擊「Sign up」或「Log in」
3. 使用 GitHub/GitLab/Bitbucket 帳號登入（或註冊新帳號）


【步驟 3】連接 Git 倉庫
--------------------------------------------------------
1. 在 Netlify Dashboard 中，點擊「Add new site」
2. 選擇「Import an existing project」
3. 選擇您的 Git 提供者（GitHub/GitLab/Bitbucket）
4. 授權 Netlify 訪問（如果是第一次）
5. 選擇包含「人類圖」項目的倉庫


【步驟 4】配置構建設置
--------------------------------------------------------
Netlify 應該會自動檢測 netlify.toml 配置

確認這些設置：
- Build command: （來自 netlify.toml，自動設置）
- Publish directory: .
- Functions directory: netlify/functions

如果都正確，直接點擊「Deploy site」


【步驟 5】等待部署
--------------------------------------------------------
等待 1-2 分鐘，直到看到「Deploy successful!」


【步驟 6】訪問網站
--------------------------------------------------------
Netlify 會提供一個網址，例如：
https://your-site-name-12345.netlify.app

點擊這個網址訪問您的網站


【步驟 7】測試功能
--------------------------------------------------------
1. 輸入出生資料
2. 點擊「計算人類圖」
3. 確認可以看到計算結果


【如果失敗】
--------------------------------------------------------
1. 查看 Netlify 構建日誌（點擊失敗的部署 → View deploy log）
2. 檢查錯誤信息
3. 清除緩存並重新部署（Deploys → Trigger deploy → Clear cache）
