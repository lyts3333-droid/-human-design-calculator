# 如何正確訪問人類圖計算器

## ❌ 錯誤的方式（會出現連接錯誤）

**不要這樣做：**
- 直接在檔案總管中雙擊 `index.html` 檔案
- 或使用 `file:///C:/Users/.../index.html` 這樣的網址

這會導致 CORS 錯誤，因為瀏覽器不允許 `file://` 協議的頁面向 `http://` 服務器發送請求。

## ✅ 正確的方式

### 步驟 1：啟動 Flask 伺服器

在命令提示符中，進入「人類圖」資料夾並運行：

```bash
cd 人類圖
python app.py
```

您應該看到：
```
* Running on http://127.0.0.1:5000
```

### 步驟 2：在瀏覽器中訪問

**打開瀏覽器，訪問：**

```
http://localhost:5000
```

或者：

```
http://127.0.0.1:5000
```

### 步驟 3：使用應用程序

現在您應該能看到完整的網頁界面，並且可以正常使用「計算人類圖」功能。

## 為什麼需要這樣做？

1. **Flask 伺服器**提供：
   - 前端頁面（`GET /`）
   - API 端點（`POST /calculate_hd`）
   - 健康檢查（`GET /health`）

2. **使用 `http://` 協議**允許：
   - 前端 JavaScript 正常發送 API 請求
   - 不會被瀏覽器 CORS 政策阻擋
   - 完整的 Web 應用體驗

## 快速檢查

如果訪問 http://localhost:5000 後仍無法連接，請檢查：

1. ✅ Flask 伺服器是否正在運行？
   - 查看命令提示符窗口，應該顯示 "Running on http://127.0.0.1:5000"

2. ✅ 瀏覽器網址列是否顯示 `http://localhost:5000`？
   - 不應該是 `file:///...` 開頭

3. ✅ 按 F12 打開開發者工具，查看控制台是否還有錯誤？

## 如果仍有問題

請提供：
- 瀏覽器網址列顯示的完整 URL
- 開發者工具控制台中的錯誤訊息
